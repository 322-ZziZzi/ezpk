EZPK War Portal v73

Mobile admin usability update based on v72.

Changes
- Admin header is no longer sticky/fixed on screens 900px and below.
- Added a compact hamburger button beside the admin brand.
- Mobile navigation opens in normal document flow and scrolls away with the page.
- Navigation closes automatically after selecting an item.
- Reduced mobile admin hero spacing.
- Added mobile-safe automatic heights, line heights, wrapping, and overflow handling.
- Increased mobile form controls to prevent browser zoom and text clipping.
- Desktop admin layout remains unchanged.
